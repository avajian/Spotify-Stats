About

What is it?

Spotify Stats is a tool that uses Spotify's API to display a user's top 10 songs from the past month, the past 6 months, and all time. It was inspired by 'Receiptify,' a popular tool for displaying a user's Spotify statistics in a receipt format. You can find it here.

How does it work?

This project uses the open-source Spotify API to fetch listening data from users and report their top songs. Unfortunately, with the data available, I’m limited to showing statistics for only the past month, the past 6 months, and all time. I don’t know how Spotify stores any of this data.

How was it made?

This project uses Python for the backend, primarily leveraging the 'Spotipy' library to simplify the data retrieval and authorization process. HTML templates are used to display the data, CSS for styling, and JavaScript for interactivity. It's hosted on PythonEverywhere, a free web hosting service. You can find the code here.
